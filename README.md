# How to run

### Install packages
- Run command `npm install`
### How to run project
- Run command `npm run dev`
### How to run unit tests
- Run command `npm run test `
## Architechture
### Feature-Based Architecture
Focus on organizing folders by features
- app: Store route folder
- components: Stroe components
- api: Store api folder
## Libraries
- @mui/material: Component Library for react
- @tanstack/react-query: Library for server state and handle data from API.
- axios: Library to send and reveive data from HTTP.
- lucide-react: Open source Lucide Icons.
